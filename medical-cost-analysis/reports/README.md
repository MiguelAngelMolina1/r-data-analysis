# Reports

This folder contains the final rendered outputs for the **Medical Cost and Lifestyle Analysis** project.

## Main Report

- **Medical_Cost_Analysis.html**  
  Final HTML report generated from the Quarto analysis file.

## Report Contents

The report includes:

- Introduction and project objectives
- Dataset overview
- Exploratory data analysis
- Univariate and bivariate visualizations
- Standard linear regression modeling
- Reduced linear regression modeling
- LASSO regression analysis
- Final interpretation and conclusions

## Summary of Findings

The report shows that **smoking status**, **BMI**, and **age** are the strongest predictors of medical insurance charges. Among these, smoking has the largest effect on annual cost, while BMI and age also demonstrate meaningful positive relationships with insurance expenses.

## Notes

This directory is intended to store final presentation-ready outputs of the project. The HTML report serves as the main deliverable for communicating results in a polished and readable format.

If the report was rendered with Quarto and includes supporting files such as images, styles, or JavaScript assets, those companion files should remain alongside the HTML output so the report displays correctly on GitHub or when opened locally.
